# Claude Tools

Custom tools and utilities for [Claude Code](https://claude.com/claude-code).

## Slash Commands

### Instruct Mode

Instruction-only mode where Claude guides you through steps without executing anything.

**Install:**
```bash
curl -fsSL https://raw.githubusercontent.com/angelcodes95/claude-tools/main/slash-commands/instruct.md -o ~/.claude/commands/instruct.md
```

**Usage:** Restart Claude Code, then type `/instruct`
